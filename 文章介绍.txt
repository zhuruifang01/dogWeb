====================================如何让猫在固定地点睡觉==================================== 
猫咪是个跳跃高手，不像狗狗如果你禁止它不许去床上睡觉，它有时候就是想去睡觉也跳不上去，猫咪就不一样了，简直是来去自如。所以要是想训练猫咪在固定的地方睡觉还是要费一番功夫的。
    <br/>
    一、准备好宠物窝
    <br/>
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;在猫咪买回来之前就要给它准备好一个宠物窝或箱子，在里面棉垫或草，如果是冬天，还要在里面放上一个热水袋或装窝放置在一个温暖的地方，让猫咪每天固定在里面睡觉。猫还是不愿在窝里睡觉，可在窝的上面盖上一个罩子使它不能往外跑，经过几次训练，猫就不会往外跑，当然也就不会钻被窝了。
    <br/>
    二、发出口令
    <br/>
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;另外，如果发现猫咪不愿意回到窝里睡觉，而跑到人的被窝进而睡时，猫咪的主人就要发出“不可以”的口令。同时把猫咪再放回到它自己的窝里，并用铁丝网做的罩子把箱子盖好，强迫猫咪在里面睡觉。这样反复训练几回，就能使猫咪逐渐养成在固定地方睡觉的良好习惯。有的猫很喜欢钻到主人的被窝里睡觉，这种习惯很不好，因为猫的一些疾病，如钱癣、弓形体病等很容易传染给人，因此训练猫不上床很有必要。
    <br/>
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;如果猫白天也喜欢上床或钻被窝，可用拍打猫的臀部，并大声训斥，将猫赶下床。由于猫对主人的情绪很敏感，这样反复多次后就可改变猫上床的坏习惯。如果这种方法不能奏效，主人还可以尝试下面的方法：事先准备2—4只玩具水枪。当发现猫上床后，主人可以站在猫看不见的地方，用吸满水的水枪向猫喷水，猫受到突如其来的袭击马上就会逃走，反复进行数次猫就不再上床了。值得注意的是切不可让猫发现人在向它喷水，否则，这种方法就难以获得良好的效果。
    <br/>
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;训练猫咪在固定的地方睡觉不是一朝一夕就可以做到的事情，猫友们一定要有耐心也要坚持，不能半途而废，不然前面训练猫咪的时间都白费了。
===============================宠物猫癣病的诊断与治疗======================================
猫癣俗称圆癣或钱癣，是由犬小孢子菌感染猫所引起一种猫的常见皮肤病，属于人畜共患病，防治该病对公共卫生安全具有一定的意义。该病在猫身上一般呈现慢性症状，虽然猫癣本身不会导致猫死亡，但是猫会由于皮肤的抵抗力下降引发严重的细菌感染，从而导致败血症、菌血症等一系列并发症，严重的话会导致动物死亡。本人在治疗猫癣上有一定的治疗经验，也取得较为满意的效果，现将诊治情况和体会与各位网友分享。
    <br/>
    一、传播途径
    <br/>
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;猫癣的传播途径是接触性传染，即该病可以通过带病猫接触的任何东西传染给健康的猫、犬及人类等。由于猫这种动物本身的活动范围很大，所以该病的传播非常难以控制，尤其是在野猫得病后。
    <br/>
    二、主要临床症状
    <br/>
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;通常情况下，得了猫癣的患猫病初无特别明显的症状，得病一段时间后出现精神不佳，病程较长的机体消瘦，毛发色泽暗淡、部分或大面积脱落。癣在躯干较多见，少数病例在头部、四肢、耳朵和尾巴也可见。患部瘙痒，轻重不一，有的猫因抓挠而继发细菌感染。典型症状是患处的皮肤脱毛，出现环形的鳞屑状，干燥圆形斑疹;有的患部还残留有部分毛根，有的患部完全脱毛，有的患部在癣斑中间生毛。有的因猫主送诊较早，需用手仔细抚摸才能发现，手触及可感觉小突起，被毛下有小块结痂和皮屑。
    <br/>
    三、猫癣的诊断方法
    <br/>
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;1、在光线较为昏暗的环境下，用伍德氏灯照在猫咪皮肤患部，可发现有黄绿色荧光的毛根。注意与部分药膏出现的绿色萤光相区别。
    <br/>
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;2、在患部皮肤边缘上刮取被毛或皮屑于载玻片上，加数滴10%氢氧化钾溶液，用酒精灯稍加热，盖上盖玻片，显微镜下观察，可见到真菌孢子。
    <br/>
    四、猫癣的治疗
    <br/>
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;采用局部用药和口服抗菌药物联合治疗。单纯用外用药不易治愈，必须口服抗菌药物治疗。
    <br/>
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;1、真泰舒喷剂：犬猫真菌性皮肤病药物，无论是治疗还是预防效果都很好。
    <br/>
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;2、克霉唑软膏：此药无刺激性，疗效不错，一般需要治疗两周左右，才能治愈。
    <br/>
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;3、酮康唑软膏：广谱抗真菌药，人和动物都能使用该药治疗癣病，连用2～8 周。
=============================养两只狗能否改善分离焦虑  ====================================
患有分离焦虑的狗狗通常比较黏人，当然也可能是这种狗比较有责任心，因为在它们看来你是它的保护对象。有的狗狗会焦虑则是因为它们的责任心过强，或者说它们把自己当成了这个家的头领。
    <br/><br/>
    患有分离焦虑的狗狗通常比较黏人，当然也可能是这种狗比较有责任心，因为在它们看来你是它的保护对象。所以当它们看不到你的身影时就会惊慌焦虑，要改善这种情况需要慢慢训练。而有人提出通过饲养两条狗狗来缓解这一情况。
    <br/><br/>
    那么养两只狗狗有助于减缓分离焦虑症呢?其实这样的做法并不能很好地改善这一问题，如果狗狗本身就有分离焦虑，第二只小狗到家里后，在错误中学习另一只狗狗的习性，那么很可能两条狗狗都有可能出现分离焦虑的问题。
    <br/><br/>
    一次养两只狗狗跟分离焦虑症没有正相关，它们很可能都会在主人离开后感到焦虑;而且如果它们过于依赖彼此，那主人在将其中一只带离时，另一只也会躁郁不安，反而不是好事情。
    <br/><br/>
    所以说，当狗狗有分离焦虑症时，最重要的就是利用训练弱化狗狗对于主人要出门的敏锐度，只要让它们知道“主人并不不是一去不回”以及“自己在家其实没什么大不了的”，狗狗习惯后就能稳定许多。
    <br/><br/>
    所以这就需要给狗狗一个习惯的过程，主人起身穿外套拿钥匙(假装要出门)后，又坐回去看电视，多练习几次，让狗狗对于准备出门的这些举动越来越无感。在出门前才给平时都不会吃的美味零食，这样狗狗就会把主人出门与获得奖励所联系起来。
    <br/><br/>
=========================================需要常做美容的犬种===============
 需要美容的狗狗以长毛狗居多，养长毛狗的人大多因为喜欢这些狗狗身上毛绒绒的感觉，象可爱的长毛玩具一样，但是要想让这些毛毛有很好的造型，就一定要去专业的美容院才行，通常自己在家是搞不定的。比如下面这几种可爱的长毛狗狗。
    <br/>
    1、英国古老牧羊犬（简称古牧）
    <br/>
    &nbsp;&nbsp;&nbsp;&nbsp;这种憨憨可爱的大个儿长毛狗，让懂狗和不懂狗的人一见都会情不自禁的爱上它们。有许多人不惜重金也要拥有一只。这种狗狗小的时候可以自己在家打理，毛毛也并不需要定期修剪，但是成年以后，如果你想让你家的古牧有型，就一定需要专业人仕帮忙了。记得以前伯爵去美容从洗澡到吹干修剪完成大约需要三到四个小时，可是更多。因为成年的古牧体重会有80斤左右，浑身的毛都十分的浓密，在家里每天梳理仍然可能会有梳不通顺的地方，所以古牧的美容，梳毛和全部吹干都占了大量的时间，到了夏天的时候还需要给它把肚子上的毛剃掉，身上的毛也要适当的修剪和打薄，因为体型大，所以整个美容的工作量很大，也难怪美容师要收较高的价钱。
    <br/>
    2、比熊
    <br/>
    &nbsp;&nbsp;&nbsp;&nbsp;超级可爱的小熊娃娃，它的毛相当的蓬松和柔软，耳朵小小的，所以人家的头可以修得圆圆的看不出耳朵，也就是著名的粉扑状。比熊属于丝毛的品种，毛非常的细，很容易打结，平时必须作好打理工作，才能到美容院节省些时间。我见过一只浑身的毛都打结的小比熊，美容师光给它打理的毛梳通就用了一个小时，小家伙也相当受罪的一直站在美容台上，相当的辛苦，所以平时每天梳毛相当重要。正常情况下，比熊美容花的时候最多的是在修剪的部分。要把它的小头修成完全的圆形，看不到耳朵，身上也要剪得齐齐的，尾巴适当的修剪，让尾毛自然的散开，这些都要仔细的一点点的剪，所以一定得耐心等待。
    <br/>
    3、贵宾
    <br/>
    &nbsp;&nbsp;&nbsp;&nbsp;贵宾是非常受欢迎的一种长毛狗狗，它有好多种颜色，几乎你想要的任何一种颜色都有，最近两年象泰迪熊一样可爱的红棕色非常时髦。贵妇的修剪可以说可以充分发挥美容师的想象力，他们把贵宾身上不同部位的毛毛剃掉，留下的毛毛剪成圓形，梳蓬松，或是染上颜色，你要是一个喜欢标新立异的人可以在贵宾身上好好体现一下你的个性。

    &nbsp;&nbsp;&nbsp;&nbsp;先重点介绍这几个吧，像可卡、西高、约克夏、西施等等一些其它种类的狗狗也是需要定期美容的，在这儿就先不一一介绍了
